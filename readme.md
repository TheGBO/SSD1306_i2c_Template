# SSD1306 I2C

## Thingy designed for kickstarting the oled display with arduino uno, platform io and c++

### Connections:

| Display Pin | Board Pin | Recommended Color (by me lol)
|-------------|-----------|-------------------------------
| GND         | GND       | Black                         
| VDD         | 5V        | Red
| SCK         | A5        | Green
| SDA         | A4        | White
