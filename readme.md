# SSD1306 I2C

## Thingy designed for kickstarting the oled display with arduino uno, platform io and c++

### Connections:

| Display Pin | Board Pin |
|-------------|-----------|
| GND         | GND       |
| VDD         | 5V        |
| SCK         | A5        |
| SDA         | A4        |
